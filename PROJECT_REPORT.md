## 1. Introduction

This project is aimed at developing a Convolutional Neural Network (CNN) model to classify malaria-infected cells in blood samples.

## 2. Objectives  
- To build a robust CNN model that achieves high accuracy in identifying malaria-infected cells.  
- To implement the model using TensorFlow and Keras frameworks.

## 3. Dataset  
The dataset used comprises thousands of images of blood cells, labeled as infected or uninfected. It comes from a publicly available source.

## 4. Methodology  
- Data Preprocessing: Images were resized, normalized, and augmented to improve model performance.
- Model Architecture: The architecture consists of several convolutional layers, max-pooling layers, and fully connected layers.
- Training: The model was trained using the Adam optimizer and categorical cross-entropy loss function.

## 5. Results  
The final model achieved an accuracy of over 95% on the validation set, demonstrating its effectiveness in classifying malaria cells.

## 6. Conclusion  
This project successfully demonstrates the capabilities of CNNs in medical image classification, providing a step towards automated disease detection.
